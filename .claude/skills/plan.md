# 実装計画作成スキル

## 使用タイミング
- 簡易仕様が確定した後
- 「計画を作って」「どう実装するか」などと言われた時

## 計画内容
以下を含める:
1. 実装ステップ（タスクの分解）
2. 依存関係（先に実装すべきもの）
3. 想定されるリスクと対策

## 出力形式
```
実装計画を作成しました：

## ステップ 1: [ステップ名]
- [具体的な作業内容]
- [使用するファイル]

## ステップ 2: [ステップ名]
- [具体的な作業内容]
- [使用するファイル]

...

## 依存関係
- ステップ 2 はステップ 1 の完了後に実行

## 想定されるリスク
- [リスク]: [対策]
```
