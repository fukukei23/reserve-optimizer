# 並列エージェント活用について

## デスクトップ版 Claude Code の並列エージェント機能

### 特徴
- **並列実行**: 複数のエージェントを同時実行可能
- **視覚的監視**: サイドバーで各エージェントの進捗を監視
- **タブ管理**: 各エージェントが別タブで管理される
- **バックグラウンド実行**: 長時間タスク中も作業継続可能

### 自動読み込みされるエージェント
`.claude/agents/` ディレクトリに配置された YAML + Markdown 形式のエージェント:
- `code-reviewer.md`: コードレビュー担当
- `decision-recorder.md`: 決定記録担当
- `tier1-validator.md`: Tier 1 タスク判定担当

### 使用方法
1. プロジェクトを Claude Desktop で開く
2. エージェントが自動読み込みされる
3. 自然言語で依頼すると、適切なエージェントが起動

### 例
```
ユーザー: 「認証機能をレビューして」
→ Code Reviewer エージェントが並列で起動

ユーザー: 「JWT認証を採用することにした」
→ Decision Recorder エージェントが自動起動
```

## 効果
- 開発速度の維持（Claude Code の対話的な開発）
- 最低限の品質担保（重要な判断の記録、コードレビュー）
- 視覚的な進捗管理
