# 安全チェックスキル

## 使用タイミング
- 実装コードを確認する前
- 「安全チェックして」「セキュリティ確認して」と言われた時

## チェック項目
### 入力検証
- ユーザー入力の検証はあるか
- SQLインジェクション対策はあるか
- XSS対策はあるか

### 認証・認可
- 適切な権限チェックはあるか
- トークンの有効期限管理はあるか

### データ保護
- 秘密情報の扱いは適切か
- エラーメッセージに機密情報が含まれないか

### 外部リソース
- 外部API呼び出しのエラーハンドリングはあるか
- レート制限の考慮はあるか

## 出力形式
```
安全チェック結果：

### OK（問題なし）
- [問題がなければ]

### 警告
- [重要度: 中] [問題点]: [対策案]

### 重大な問題
- [重要度: 高] [問題点]: [対策が必要]

### 推奨対応
- [改善の余地がある箇所]
```
