## 2026-03-30

### GitHub PAT共有の危険性问题（02:18-02:21 UTC）
- **事象**: ふくけいから「GitHubのPersonalAccessTokenはDiscordに共有危险？」と質問
- **クロタム回答**: GitHub PAT（`ghp_`から始まるトークン）をDiscordに絶対に共有しない
- **理由**:
  1. Discordの会話履歴は平文保存される
  2. Claudeの会話履歴にも残る（学習データやログに含まれるリスク）
  3. トークン一つでリポジトリ読み書き全権限にアクセス可能

**結論**: PATを共有するのではなく、環境変数や設定ファイルで管理すること

### NexusCoreリポジトリ可視性问题（2026-03-29夜）
- **事象**: ふクロウ（VPS上のフクロウ）ではNexusCoreが見えないが、クロタム（Surface Go/Claude Code）では見える
- **原因**: 認証トークンが異なる
  - クロタム: `gh auth login`済み（Surface Go上）
  - フクロウ: `GITHUB_TOKEN`環境変数（VPS上）
- **解決方法**: 別エージェントのリポジトリ読み書きは、そのエージェント経由でしかできない
