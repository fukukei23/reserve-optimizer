```bash

  --build-arg OPENCLAW_INSTALL_BROWSER=1 \
```


```bash

```

## 2026-03-21

### 「All models failed」エラーの正しい理解（重要訂正）
**ふくけいによる公式説明（00:40 UTC）**:

従来の解釈（フクロウの誤解）:
- 「全モデルが同時故障」「rate limit到達」

**正しい状態**:
- モデル故障ではない → **推論がタイムアウト时间内に終わらない**
- 1回タイムアウトすると → **その後も回復しない**（推論が完了しても回答不能のまま）

**問題の本質**:
1. モデルの処理速度 < タイムアウト設定 → タイムアウト発生
2. その後、何故か回復しない → 連鎖的に失敗が続く
3. これは「前からある問題」

**確認すべきこと**:
- `timeoutSeconds` 設定値（openclaw.json）
- タイムアウト後の回復机制の有無
- アーキテクチャレベルの問題の可能性

**ステータス**: 調査待ち
