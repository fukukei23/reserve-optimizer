# コスト試算スキル

## 計算式

```
コスト = (入力トークン ÷ 1,000,000 × $1) + (出力トークン ÷ 1,000,000 × $3.2)
```

## 価格表（参考）

| 入力 | 出力 | コスト |
|------|------|--------|
| 300 | 400 | $0.00158（約0.25円） |
| 1,000 | 1,000 | $0.0042（約0.67円） |
| 10,000 | 10,000 | $0.042（約6.7円） |
| 100,000 | 100,000 | $0.42（約67円） |

## 判断基準

| 月間コスト | 判定 |
|-----------|------|
| ~3,000円 | OK |
| 3,000~10,000円 | 要確認 |
| 10,000円~ | 承認必須 |

## 出力フォーマット

```
【コスト試算】
- 入力: ~{N}トークン
- 出力: ~{N}トークン
- 概算: ${cost}（約{円}円）
```

## スクリプト使用

```bash
python3 scripts/calc_cost.py <入力トークン> <出力トークン>
```
