# Reviews ディレクトリ

- `daily-flash-YYYY-MM-DD.md` … 長尺5分版の台本
- `shorts-YYYYMMDD-captions.json` / `.srt` / `.ttml` … Shorts字幕
- `daily-flash-draft.json` … Summarizerからの生出力（テンプレ差し込み前）

## Discord投稿テンプレ
```
🦉 Claw5 Flash #{{num}}
- 長尺台本: reviews/daily-flash-{{date}}.md
- Shorts字幕: reviews/shorts-{{date}}.srt
- Hook: {{hook}}
```

## レビュー手順
1. 台本を貼る（コードブロック）
2. 変更を箇条書きでコメント
3. OKなら ✅ リアクション
4. 収録完了後、同じスレに公開リンクを追記
