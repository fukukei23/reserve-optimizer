# よくある質問（各400字以内・ココナラ画面へ）

**Q1：月額費用はかかりますか？**
いいえ、完全買い切りです。サービス価格のみで制作し、月額の利用料や保守料は一切かかりません。GAS（Google Apps Script）は無料枠が広く、小規模店舗の予約受付なら無料範囲で運用できます。公開後の軽微な変更も別途ご相談いただけます。

**Q2：LINE公式アカウントを持っていなくても大丈夫ですか？**
はい、お持ちでなくても構いません。開設手順を丁寧にご案内いたします。LINE公式アカウントは月50件までのメッセージ送信なら無料でご利用いただけます。予約受付の自動応答であれば、無料枠で十分運用可能です。

**Q3：予約を受ける際、お客様のスマホ画面はどうなりますか？**
LINEのトーク画面で、メニュー選択や日時指定をボタン形式で進められます。新たなアプリのインストール不要で、LINEさえあればどなたでも簡単に予約できます。

**Q4：決済やGoogleカレンダーとの連携はできますか？**
はい、オプションで対応可能です。決済（Stripe連携）による予約時の前払い、Googleカレンダー連携による担当者ごとのスケジュール同期、キャンセル待ち、リッチメニューの追加など、店舗の運用に合わせて拡張できます。

**Q5：パソコンやITの知識がなくても運用できますか？**
はい。設定や更新の操作手順は分かりやすくご説明し、納品後もサポートいたします。予約状況はスプレッドシートに自動記録されるため、Excel感覚で確認できます。

**Q6：予約データはどこに保存されますか？**
Googleスプレッドシートに自動保存されます。クラウド上で管理されるため紛失の心配がなく、複数人での共有やバックアップも容易です。
